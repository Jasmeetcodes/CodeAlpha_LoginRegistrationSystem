# CodeAlpha - Login & Registration System

## Project Description

This project is a **Login and Registration System** developed in **C++** as part of the **CodeAlpha C++ Programming Internship**.

The application allows users to register with a unique username and password, securely stores user credentials in a text file, and enables registered users to log in using their credentials.

---

## Features

- User Registration
- User Login
- Duplicate Username Validation
- File Handling using `users.txt`
- View Registered Users
- Menu-Driven Interface
- Object-Oriented Programming (OOP)

---

## Technologies Used

- C++
- File Handling (`fstream`)
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)

---

## How to Run

### Compile

```bash
g++ login_registration.cpp -o login
```

### Run

For Windows:

```bash
.\login.exe
```

For Linux/macOS:

```bash
./login
```
